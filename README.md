# Elevator CMS Web Application

A modern, premium monitoring dashboard for elevator management systems.

## Tech Stack

- **Framework**: React 19 + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Components**: Shadcn UI (Manual integration)
- **Package Manager**: pnpm

## Getting Started

### Prerequisites

Ensure you have [pnpm](https://pnpm.io/) installed.

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

## Project Features

- **Real-time Monitoring**: Dashboard with a clear overview of elevator statuses.
- **Role-Based Access**: Mock authentication for Admins, Operators, and Viewers.
- **Modern UI**: Clean design with Tailwind v4 and premium aesthetics.
